# Eleven Sky - Anonymous AI Video Chat

A premium futuristic anonymous video chat platform with AI features, built with WebRTC, Socket.IO, and Node.js.

## Features
- Glassmorphism UI
- WebRTC Video/Audio Chat
- Real-time Text Chat
- AI Moderation (simulated)
- Anonymous Matching

## Setup

1. `npm install`
2. `npm start`
3. Open http://localhost:3000

## Tech Stack
- Frontend: HTML5, CSS3 (Glassmorphism), ES6
- Backend: Node.js + Express + Socket.IO
- Video: WebRTC

GitHub-ready and production scalable.
